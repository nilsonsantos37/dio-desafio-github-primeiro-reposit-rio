Clonando repositório do github para a minha maquina: abrir terminal, digitar git clone e colar o link 
que foi copiado do repositório.
Para entrar em uma determinada pasta pelo terminal basta digitar: cd + o caminho do repositório se aparecer (man)
vc estará dentro da pasta.

Para inserir um arquivo da minha máquina local, Para o servidor do GitHub, basta abrir o 
terminal do gitBash e digitar:
git status para verificar se o git reconheceu que há uma nova pasta para inserir no servidor do git.
 Para adicionar a nova pasta para o controle de Versão bata digitar: git add . ou git add -A para adicionar todos 
os arquvos. Depois digita git status para verificar se houve a alteração. aqui só conseguimos adicionar 
o arquivo no repositório da máquina local  
 

 Incluindo o arquivo dentro do servidor github
no terminal digitar git commit -m  "comentário coerente e com sentido ". o comit precisa ter um comentário coerente 
 e com sentido dentro do parenteses para que o arquivo que está sendo enviado.
 novamente digita  git status para ver a mensagem dizendo que preciso dar um gip push para enviar os 
arquivos para a nuvem.
dentro do gitHub na pasta criada precione F5 para atualizar a pasta com as novas anotações ou alterações 
feitas lá ma maquina local.
